# 🇲🇲 Myanmar Quiz Game
### Test Your Knowledge About Myanmar!
**By Nyein Sat Thaw**

---

<!-- slide 2 -->
# The Problem
- Many people don't know enough about Myanmar
- Learning about a country should be fun
- No simple, accessible quiz app existed

---

<!-- slide 3 -->
# What I Built
- A web-based quiz game about Myanmar
- 4 categories: History, Culture, Geography, Food
- 40+ questions, score tracking, beautiful UI

---

<!-- slide 4 -->
# How I Built It
- **MCP**: Connected filesystem tools for project management
- **Skill**: Created quiz-generator skill for making new questions
- **Agent**: Built quiz-reviewer agent to validate answers
- All built with **Claude Code** 🤖

---

<!-- slide 5 -->
# Why It Matters
- Educates people about Myanmar's rich culture
- Fun way to learn history, geography, food
- Open source — anyone can contribute questions
- Accessible — runs in any web browser
